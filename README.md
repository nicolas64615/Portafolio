💻 Portafolio Personal – Nicolás Bustamante Jara

📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **portafolio personal web** creado con HTML5, CSS3 y GitHub, en el contexto 
del módulo de Desarrollo Web Front-End.

La página es una **Single Page Application (SPA)** que presenta mi perfil profesional, habilidades, proyectos 
y un formulario de contacto, aplicando buenas prácticas de desarrollo web y diseño responsivo.

---

🎯 Objetivo

Demostrar conocimientos en:

* Estructura semántica con HTML5
* Estilización avanzada con CSS3
* Uso de Flexbox y Grid
* Diseño responsive
* Control de versiones con GitHub

---

 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Git y GitHub

🧩 Secciones del Sitio

* 🔹 **Header:** Logo + barra de navegación
* 🔹 **Sobre mí:** Información personal y perfil profesional
* 🔹 **Habilidades:** Tabla de tecnologías + listas
* 🔹 **Proyectos:** Galería con imágenes y enlaces
* 🔹 **Contacto:** Formulario con validación HTML5
* 🔹 **Footer:** Información de derechos y créditos

---

🎨 Características

✔ Uso de etiquetas semánticas (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
✔ Diseño moderno y limpio
✔ Uso de **Flexbox** para layout (navbar y distribución)
✔ Uso de **CSS Grid** para galería de proyectos
✔ Página **responsive** (adaptable a móvil y tablet)
✔ Formulario estilizado
✔ Tabla con estilos personalizados

---

📱 Diseño Responsive

El sitio está optimizado para:

* 📱 Móviles (max-width: 480px)
* 📲 Tablets (max-width: 768px)
* 💻 Escritorio
